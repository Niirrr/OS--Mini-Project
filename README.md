# 🖥️ Mini OS Simulator

An interactive **Operating System Simulator** developed using **Python** and **ttkbootstrap**. The application provides a visual way to understand CPU scheduling algorithms through an intuitive graphical interface, Gantt chart visualization, and performance metrics.

## ✨ Features

- Interactive CPU Scheduling Simulator
- Gantt Chart visualization
- Waiting Time and Turnaround Time calculation
- Simple and user-friendly interface
- Modular project structure

## ⚙️ Algorithms Implemented

- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Priority Scheduling
- Round Robin (RR)

## 🛠️ Built With

- Python 3
- Tkinter
- ttkbootstrap

## 📂 Project Structure

```text
MiniOSSimulator/
├── algorithms/          # CPU scheduling algorithms
│   ├── fcfs.py
│   ├── sjf.py
│   ├── priority.py
│   └── rr.py
├── assets/              # Images and resources
├── gui/                 # GUI pages
├── models/              # Data models
├── main.py              # Application entry point
└── README.md
```

## 🎯 Purpose

This project was developed as part of an **Operating Systems Semester Project** to provide a practical understanding of CPU scheduling algorithms through interactive visualization.
