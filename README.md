# 🖥️ Mini OS Simulator

> An interactive Operating System simulator built with **Python** and **ttkbootstrap**, designed to visualize CPU scheduling algorithms through an intuitive graphical interface.

## 📖 Overview

Mini OS Simulator is an educational desktop application that demonstrates core Operating System concepts through interactive visualizations. Instead of simply displaying numerical outputs, the simulator allows users to enter process information, execute scheduling algorithms, and observe the execution order through dynamically generated Gantt charts and performance metrics.

The application is designed with a modern, user-friendly interface using **ttkbootstrap**, making it suitable for learning, demonstrations, and academic presentations.

---

## ✨ Features

- Modern dark-themed graphical user interface
- Interactive CPU Scheduling simulation
- Dynamic process creation and management
- Automatic Gantt Chart generation
- Performance metric calculation
- Responsive desktop interface
- Modular project architecture for future expansion

---

## ⚙️ CPU Scheduling Algorithms

The simulator currently supports:

- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Shortest Remaining Time First (SRTF)
- Priority Scheduling
- Round Robin (RR)

For each simulation, the application displays:

- Execution Order
- Gantt Chart
- Waiting Time
- Turnaround Time
- Completion Time
- Average Waiting Time
- Average Turnaround Time

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/<your-username>/Mini-OS-Simulator.git
```

### Navigate to the project

```bash
cd Mini-OS-Simulator
```

### Install dependencies

```bash
pip install ttkbootstrap
```

### Run the application

```bash
python main.py
```
